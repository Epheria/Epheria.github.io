## Unreal Mac OSでRiderを使ってライブコーディングする方法

- Mac OS (M4)でRiderを使ってライブコーディングをする方法をいくらググっても見つけられなかったので、自分で探しました。

<br>

#### 1. Preference 設定

- Macでは、Unreal Preference設定はMac上部のツールバーで確認できます。 **ショートカットキー：Cmd + ,**

![Desktop View](/assets/img/post/unreal/unrealmac01.png){: : width="500" .normal }    

<br>

![Desktop View](/assets/img/post/unreal/unrealmac02.png){: : width="500" .normal }    


- **Source Code** に入り、 **Rider UProject** に設定します。

<br>

#### 2. Toolで Refresh Rider Uproject Project を実行させます。

![Desktop View](/assets/img/post/unreal/unrealmac04.png){: : width="500" .normal }    

- **Rider UProject** は開いている必要があります。

<br>

#### 3. Unrealプロジェクト下部ツールバーを見るとグリッド模様のアイコンが見えますが、こいつをクリックするとライブコーディングが実行されます...

![Desktop View](/assets/img/post/unreal/unrealmac03.png){: : width="500" .normal }    

- こんなところに隠れていたなんて... ショートカットキーを設定する方法がないかも探してみる必要がありそうです...
