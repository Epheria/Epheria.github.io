## Enabling Live Coding with Rider on Unreal Mac OS

- I couldn't find a way to use Rider for Live Coding on Mac OS (M4) no matter how much I Googled, so I found it myself.

<br>

#### 1. Preference Settings

- On Mac, Unreal Preference settings can be found in the top toolbar. **Shortcut: Cmd + ,**

![Desktop View](/assets/img/post/unreal/unrealmac01.png){: : width="500" .normal }    

<br>

![Desktop View](/assets/img/post/unreal/unrealmac02.png){: : width="500" .normal }    


- Go to **Source Code** and set it to **Rider UProject**.

<br>

#### 2. Run "Refresh Rider Uproject Project" in Tools

![Desktop View](/assets/img/post/unreal/unrealmac04.png){: : width="500" .normal }    

- **Rider UProject** must be open.

<br>

#### 3. Click the grid icon in the bottom toolbar of the Unreal Project to run Live Coding...

![Desktop View](/assets/img/post/unreal/unrealmac03.png){: : width="500" .normal }    

- I can't believe it was hidden here... I'll have to look for a way to set a shortcut key...
