# Excalidraw Design Guide

이 문서는 Excalidraw 다이어그램의 시각적 품질을 일관되게 유지하기 위한 디자인 레퍼런스다.
다이어그램 생성 전 `read_diagram_guide`를 호출하면 최신 가이드를 확인할 수 있다.

## 색상 팔레트 (Excalidraw 공식)

의미별로 색상을 매핑하면 다이어그램을 처음 보는 사람도 직관적으로 이해할 수 있다.
한 다이어그램에 3~4색 이내로 사용해야 산만해지지 않는다.

## 사이징 규칙

너무 작은 도형은 텍스트가 넘치고, 너무 큰 도형은 공간 낭비다. 아래 최솟값을 지키면 대부분 깔끔하게 나온다.

- 최소 도형: width >= 120px, height >= 60px
- 폰트: 제목 >= 20, 본문 >= 16, 라벨 >= 14
- 도형 간격: 40~80px
- 화살표 최소 길이: 80px
- 20px 그리드 정렬 권장

## 레이아웃 패턴

- **가로(LR) 레이아웃 우선** — 대부분의 플로우/아키텍처는 좌→우가 가독성이 좋다
- Zone 배경: 큰 rectangle + 낮은 opacity(35~50) + roughness: 2
- Zone 라벨: 좌상단에 작은 폰트(16px)로 배치
- 화살표: 반드시 `startElementId`/`endElementId` 바인딩 — 수동 좌표로 하면 도형 이동 시 끊김

## 생성 순서

이 순서로 요소를 배치하면 바인딩이 깨지지 않고, 레이어 겹침도 자연스럽다:

1. `clear_canvas` — 캔버스 초기화
2. 배경 Zone (큰 사각형, 연한 fill, 낮은 opacity)
3. 주요 노드 (도형 + 텍스트, id 부여)
4. 화살표 (바인딩 ID로 연결)
5. 주석 (독립 텍스트 — 제목, 설명, 각주)

## 스타일 속성 요약

| 속성 | 값 | 설명 |
|------|-----|------|
| `fontFamily` | `5` (Excalifont) | CJK(한중일) 포함 필기체. 모든 텍스트에 기본 사용 |
| `roughness` | `1~2` | 손그림 테두리 — Excalidraw다운 느낌의 핵심 |
| `roughness` | `0` | 점선/대시 라인에만 사용 (선이 흐트러지면 대시가 안 보임) |
| `strokeWidth` | `2` | 기본, 강조 시 `3` |
| `opacity` | `35~50` | Zone 배경에만 적용 |

> **fontFamily 참고**: Virgil(1)은 구형으로 CJK 글리프가 없어 한글이 시스템 폰트로 폴백된다.
> Excalifont(5)는 프론트엔드 기본값이기도 해서 별도 린트가 필요 없다.
